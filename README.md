This repository contains a structured sales dataset and an Excel-based analytics dashboard used to analyze **global revenue, profitability, channels, products, and discount behavior**.

It supports interactive business intelligence using PivotTables, PivotCharts, and slicers only in Excel.

## Contents

- `Excel_Assignment and Brief (Done).xlsx`  
  - Inclusive of:
  - Raw sales data
  - Staging Table
  - Cleaned & structured tables
  - PivotTables and PivotCharts
  - Interactive dashboard with slicers
  - Dashboard Insight

## Key Fields in the Dataset

- `Order Date`
- `Region`, `Country`
- `ProductCategory`, `SKU`
- `Salesperson`
- `Channel` (Direct, Distributor, Online, Retail, etc.)
- `GrossRevenue`, `UnitCost`, `GrossProfit`
- `Discount %`
- `On-Time Delivery %`

## Dashboard Features

- KPI summary:
  - Total Revenue
  - Gross Profit
  - Margin %
  - Average Order Value
  - On-Time %

- Visualizations:
  - Revenue Trend by Month
  - Profit by Region & Channel
  - Top 10 SKUs by Revenue
  - Discount % Distribution & Outlier Detection

- Interactivity:
  - Slicers for Region, Country, ProductCategory, Salesperson, Channel, and Month


## Use Cases

- Business performance reporting
- Sales and marketing analysis
- Pricing and discount optimization
- Portfolio/case study demonstration

## Tools Used

- Microsoft Excel
  - PivotTables & PivotCharts
  - Slicers
  - Scrolls for what-if analysis
